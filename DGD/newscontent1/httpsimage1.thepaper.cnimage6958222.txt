<h1 class="news_title">巴塞尔表展在即，这些抢先登场的腕表将是市场风向标（下）</h1>

<div class="news_txt" data-size="standard"><u>距离3月22日2018年巴塞尔世界钟表珠宝博览会（Baselworld 2018）在瑞士巴塞尔拉开帷幕已经不到一个月的时间了。预计近1500个品牌、15万参观者和超过3000名媒体记者将齐聚巴塞尔这座小城，共同迎来这个世界规模最大的钟表珠宝博览会。 <br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="396" src="http://image.thepaper.cn/www/image/6/958/220.jpg" width="600"/></div></u><span style="color: rgb(153, 153, 153);">Baselworld已成为了有着百年历史的珠宝钟表业最盛大的集会。<br/><div class="contheight"></div></span>作为世界最大的钟表珠宝展，Baselworld由来已久，其历史最早可以追溯到1917年。发展到今天，Baselworld已成为了有着百年历史的珠宝钟表业最盛大的集会，各大品牌都会借此时机发布当年的新品，而来自世界各地的经销商、收藏家和媒体也因此汇聚当地，领略业界最新消息。可以说，巴塞尔世界钟表珠宝博览会对于未来一年全球珠宝腕表市场有着举足轻重的影响。<br/><div class="contheight"></div>过去的一年对于瑞士制表业来说是看见曙光的一年，在经历了一系列的市场冲击之后，2017年年底的市场销售略有回升，也让很多品牌开始积极地调整新一年的市场策略。那么，为了将这样的好势头延续下去，各家品牌又拿出了哪些代表作品来占领市场呢？澎湃新闻记者为你收集了眼下提前亮相的巴塞尔表展新品，这些腕表也许就是2018年的最抢眼的腕上明星。<br/><div class="contheight"></div><strong>宝珀：历法大师的全新演绎<br/><div class="contheight"></div></strong>自2002年首次问世，两地时全历月相功能便成为宝珀的经典作品。2018年巴塞尔上，宝珀更将推出全新两地时全历月相腕表，搭载品牌专属隐藏式调校按钮，对实用的两地时全历月相复杂功能再次诠释。<br/><div class="contheight"></div>全历月相腕表不仅可以显示星期和月份，还能通过蓝钢蛇形指针读取表盘内圈的日期指示刻度；六点钟位置的月相视窗，堪称品牌的经典标识。此外在两地时全历月相腕表上，两地时（GMT）功能的添加，使此款时计跻身复杂功能腕表的行列，成为跨时区商务人士理想的出行伴侣——时针指示当地时间，另一根红尖指针显示旅行者家乡所在地的第二时区时间。 <br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="355" src="http://image.thepaper.cn/www/image/6/958/211.jpg" width="600"/></div><span style="color: rgb(153, 153, 153);">宝珀Villeret经典系列全新两地时全历月相腕表表盘布局、表背隐藏式调校按钮与金质摆陀<br/><div class="contheight"></div></span>如今的宝珀自产机芯亦搭载了安全可靠的硅游丝，保证腕表高防磁和高精准的卓越性能。宝珀专属的隐藏式调校按钮，佩戴者只需借助指尖，便可轻松设置全历和月相显示，无需再使用额外的调校工具。通过表冠，还能快速调校当地时间，每次调校的单位为一小时。<br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="473" src="http://image.thepaper.cn/www/image/6/958/212.jpg" width="600"/></div><span style="color: rgb(153, 153, 153);">宝珀Villeret经典系列两地时全历月相腕表红金款和精钢款<br/><div class="contheight"></div></span><strong>宝齐莱：130周年的献礼<br/><div class="contheight"></div></strong>今年是宝齐莱130周年，品牌为此准备了一系列极具纪念意义的新作。尤其值得一提的是，宝齐莱为自主研发的CFB A2050外缘摆陀上链机芯披上了华贵的新外衣：全新43毫米马利龙缘动力腕表（Manero Peripheral）以玫瑰金表壳亮相，将品牌首个外缘摆陀技术烘托得更加出众。<br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="434" src="http://image.thepaper.cn/www/image/6/958/210.jpg" width="600"/></div><span style="color: rgb(153, 153, 153);">玫瑰金马利龙缘动力腕表<br/><div class="contheight"></div></span>此新作保留马利龙缘动力腕表系列的典雅浑圆外型，其鲜明轮廓、柔和弧度及抛光与哑面打磨的质感对比。虽然表壳尺寸更为宽大，但依旧保持了恰到好处的线条比例，表盘的楔形时标及尖剑形指针大小亦随表壳尺寸变化而调整，指针滑过时标及刻度时更觉顺畅流丽。 <br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="444" src="http://image.thepaper.cn/www/image/6/958/209.jpg" width="300"/></div><span style="color: rgb(153, 153, 153);">玫瑰金马利龙缘动力腕表<br/><div class="contheight"></div></span>全新马利龙缘动力腕表与系列现有表款一样，装配宝齐莱自家设计及研制的CFB A2050自动上链机芯，这机芯是CFB A2000机芯系列的延伸。CFB A2050机芯拥有瑞士官方天文台认证，备有中置时针及分针、6时位小秒针以及3时位日期显示等功能。CFB A2050机芯由宝齐莱独有的双向转动自动上链摆陀，驱动上链储能，机芯的日内瓦条纹打磨修饰精美，倍添优雅气质；机芯动力储存约55小时，性能可靠，精密准绳。<br/><div class="contheight"></div><strong>宇舶：红色魔力<br/><div class="contheight"></div></strong>“别具一格，独一无二”是HUBLOT宇舶表的制表理念。此次，宇舶表再度创新，推出全球首个明亮红色陶瓷新材质腕表。这一弹性出众的全新陶瓷材质拥有高密度的特性，从而使其硬度达到1500HV2，远超传统陶瓷1200HV2的硬度。全新彩色陶瓷的问世源于重大创新，该技术将高压力与高热相融合，无需烧制颜料就能锻造陶瓷。凭借这种独特的着色技巧，宇舶表将带来一系列彩色陶瓷腕表。 <br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="447" src="http://image.thepaper.cn/www/image/6/958/214.jpg" width="300"/></div><span style="color: rgb(153, 153, 153);">全新科技打造的高科技陶瓷，不仅色彩明艳，更具有超强硬度。<br/><div class="contheight"></div></span>首发的这款红色魔力陶瓷Big Bang Unico腕表采用富有活力的明亮红色，与表带相呼应。限量发售500枚，表壳直径45mm，表圈由宇舶表专利的红色陶瓷制成，且经过抛光处理。凸缘、刻度盘、分数和秒数计数器、阿拉伯数字和指针均采用同样明亮红色，与表圈色调相匹配，突显了与Unico HUB124自产机芯和导柱轮机械性的差别，并藉由蓝宝石镜面表盘清晰可见。<br/><div class="contheight"></div><strong>真力时：古巴风情<br/><div class="contheight"></div></strong>2018年，适逢第二十届古巴著名的哈瓦那雪茄节，瑞士高级制表品牌ZENITH真力时与传奇雪茄品牌制造商Cohiba重磅出击，联合推出别具风格的限量腕表，为真力时飞行员系列Type 20家族增添了两位全新成员——全球限量发售50枚的飞行员系列 Type 20 Cohiba 马杜罗5号玫瑰金特别版腕表及150枚由青铜打造的飞行员系列 Type 20 Cohiba 马杜罗 5号特别版计时腕表。 <br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="457" src="http://image.thepaper.cn/www/image/6/958/213.jpg" width="600"/></div><span style="color: rgb(153, 153, 153);">飞行员系列 Type 20 Cohiba 马杜罗5号玫瑰金特别版腕表和飞行员系列 Type 20 Cohiba 马杜罗 5号特别版计时腕表<br/><div class="contheight"></div></span>真力时与古巴雪茄品牌Cohiba的合作关系始于2016年，今年适逢Cohiba品牌成立五十周年。今年真力时与Cohiba携手推出了两款复古格调的限量表款——飞行员系列 Type 20 Cohiba 马杜罗5号玫瑰金特别版腕表和飞行员系列 Type 20 Cohiba 马杜罗 5号特别版计时腕表。两款腕表防水性能达到10个大气压，并拥有50小时的动力储备。沿袭真力时飞行员腕表的经典特征，如霸气表壳、醒目易调节的棘轮式表冠以及腕表超大荧光数字，与此同时，也结合了代表马杜罗5号雪茄的设计细节，丰富质感尽显。<br/><div class="contheight"></div><strong>豪雅：新一代经典<br/><div class="contheight"></div></strong>泰格豪雅卡莱拉系列问世半世纪以来经历各种不同形式的变化和技术改进，终于成为典范。与生俱来的先锋加上深植其中的品牌基因，从1960年代的极简表款到近几年推出机械表和智能腕表，在每个时代写下历史。兼具运动风格与现代感，新一代泰格豪雅卡莱拉系列Heuer 02自动计时码表有着21世纪的卡莱拉系列时尚线条，并通过模块化表壳和速度计表圈表现与众不同的一面。 <br/><div class="contheight"></div><div style="text-align:center;"><img alt="" height="546" src="http://image.thepaper.cn/www/image/6/958/207.jpg" width="600"/></div><span style="color: rgb(153, 153, 153);">新一代泰格豪雅卡莱拉系列Heuer 02自动计时码表有着21世纪的卡莱拉系列时尚线条，并通过模块化表壳和速度计表圈表现与众不同的一面。<br/><div class="contheight"></div></span>为了庆祝此次周年纪念，新一代的泰格豪雅卡莱拉系列配备全新机芯Heuer 02 Calibre，同时更回归1963年的3-6-9计时盘设计。分钟和小时计时盘分别设于3点钟和9点钟位置，小秒盘位于6点钟位置，再加上4:30位置的日期窗口。镂空表盘结合夜光柱状指针和时标，嵌入防水性能达100米的43毫米模块化表壳中，推出多种不同版本，包括精钢、碳纤维、陶瓷或玫瑰金。共13款新作，其中一款配备GMT功能，搭配橡胶、皮革、金属或陶瓷表带和折叠表扣。 <br/><div
